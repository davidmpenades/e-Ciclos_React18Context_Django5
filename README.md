# e-Move
# Alquiler de Bicicletas Eléctricas

¡Bienvenido a nuestra aplicación de gestión de alquiler de bicicletas eléctricas! Esta aplicación te permite alquilar bicicletas eléctricas de manera sencilla y eficiente para disfrutar de la movilidad urbana de una manera más ecológica y saludable.

## Tecnologías utilizadas

[![My Skills](https://skillicons.dev/icons?i=react,js,tailwind,html,css,django,py,postgres,docker)](https://skillicons.dev)

## Características principales

- <b>Página de inicio: </b> Aquí encontrarás una breve explicación sobre los beneficios de la movilidad urbana con bicicleta eléctrica.
- <b>Página de alquiler: </b> Ofrecemos una guía detallada sobre cómo utilizar nuestra aplicación de alquiler de bicicletas eléctricas. Además, incluimos un mapa interactivo donde puedes encontrar nuestras estaciones. Al hacer clic en una estación,
   obtendrás información sobre el nombre de la estación, las bicicletas disponibles y los anclajes libres.
- <b>Página de planes: </b> Descubre los dos planes de alquiler entre los que puedes elegir según tus necesidades y preferencias.
- <b>Página de contacto: </b> Ofrecemos un servicio de contacto directo con el administrador a través de correo electrónico, utilizando email.js para una comunicación rápida y eficiente.
- <b>Página de login y registro: </b> Para acceder a todas las funcionalidades de la aplicación, puedes registrarte y luego iniciar sesión para gestionar tus alquileres y obtener acceso a funciones adicionales.
- <b>Dashboard del administrador: </b> Los administradores tienen acceso a un panel de control donde pueden gestionar todas las estaciones, bicicletas, anclajes, usuarios y incidencias. Pueden agregar, modificar o eliminar estaciones y bicicletas, así como cambiar el estado
   de las incidencias para mantener a los usuarios informados.
